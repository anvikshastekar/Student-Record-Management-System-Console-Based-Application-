# Student-Record-Management-System-Console-Based-Application-
📘 Student Record Management System (Console-Based Application)  The Student Record Management System is a menu-driven Python application designed to manage student academic records efficiently using basic data structures and control flow concepts.
This project allows users to add, view, search, update, and analyze student data directly from the console. Each student record is stored as a dictionary, and all records are maintained in a list, demonstrating practical use of lists, dictionaries, loops, functions, and conditional statements.

🚀 Features

➕ Add new students with marks in 5 subjects

👀 View complete details of all students

🔍 Search student by Roll Number

📊 Display class statistics (average, highest, lowest scorer)

📝 Update student marks

📈 Sort students by total marks

🏆 Automatic grade calculation (A–F)

🚫 Prevent duplicate roll numbers

📊 Grade distribution analysis

🔄 Continuous menu-driven execution until exit

🛠 Concepts Used

Python Lists

Dictionaries

Functions

While Loop (Menu-driven system)

For Loop (Searching & Displaying)

Conditional Statements

Lambda Function (Sorting)
